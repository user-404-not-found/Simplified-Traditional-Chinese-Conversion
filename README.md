# Simplified-Traditional-Chinese-Conversion
全自動簡繁快捷轉換助手 
這是一個專為 Windows 使用者設計的輕量化文字轉換工具。無需切換視窗，只需一組熱鍵，即可自動識別並轉換選取的文字。

核心功能
自動識別轉換：聰明判斷來源文字是簡體或繁體，並自動轉換為目標語系。
全域熱鍵支持：按下 Ctrl + Shift + Q 立即轉換，支援瀏覽器、記事本及各類編輯器。
常駐系統托盤：介面簡潔，可隨時透過右下角圖示手動切換「強制模式」。
台灣用語優化：採用 s2twp 轉換標準，更貼近台灣使用者的語言習慣。

安裝與使用說明
下載：下載下方的 全自動簡繁快捷轉換助手.exe。
執行：由於本軟體需要「監聽熱鍵」並「模擬按鍵（Ctrl+C/V）」，請按右鍵選擇「以系統管理員身份執行」，否則在部分視窗中可能失效。
操作：選取任何文字，按下 Ctrl + Shift + Q 即可轉換簡體或繁體

安全提示
殺毒軟體報錯：本軟體由 Python 編寫並打包。由於具備熱鍵監聽（Keyboard Hook）功能，部分防毒軟體（如 Windows Defender）可能會誤報。本計畫代碼完全開源，請放心使用。
隱私承諾：本工具僅於本地執行，不會將任何剪貼簿內容上傳至伺服器。
